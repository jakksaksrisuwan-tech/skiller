# Practice — sum_even

Verifies your environment works end-to-end.

## Task
Implement `sum_even(nums)` that returns the sum of the even numbers in `nums`.

## Examples
```python
sum_even([1, 2, 3, 4])  # -> 6
sum_even([])            # -> 0
sum_even([1, 3, 5])     # -> 0
sum_even([-2, -1, 0])   # -> -2
```

Edit `solution.py` with `e` (vim) — save & quit (`:wq`) — then `t` to run tests.
